# DetectorBox

DetectorBox é um sistema para o controle de qualidade em linhas de produção de Bebidas. Por meio de visão computacional o sistema pode detectar a ausencia de garrafas em garrafeiras na linha de produção e rejeitar as garrafeiras que não estiverem conformes.

## Recursos

- Sistema de Visão Computacional, usando OpenCV
- Arquitetura extensível baseada em classes abstratas
- Fácil integração com novos protocolos

## Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/DetectorBox.git
cd DetectorBox
```

Instale as dependências (se houver):

```bash
pip install -r requirements.txt
```
